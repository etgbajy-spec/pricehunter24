🔧 캐시 문제 해결 가이드

⚠️ 현재 문제

admin-dashboard.html의 스크립트가 실행되지 않고 있습니다. 이는 브라우저 캐시 문제일 가능성이 높습니다.

1단계: 강력 새로고침

Ctrl + Shift + R (Windows) 또는 Cmd + Shift + R (Mac)

2단계: 개발자 도구에서 캐시 비활성화

  1. F12 키를 눌러 개발자 도구 열기
  2. Network 탭으로 이동
  3. "Disable cache" 체크박스 선택
  4. 개발자 도구가 열린 상태에서 새로고침

3단계: 브라우저 캐시 완전 삭제

Chrome: 설정 → 개인정보 보호 및 보안 → 인터넷 사용 기록 삭제

Edge: 설정 → 개인정보, 검색 및 서비스 → 검색 데이터 지우기

4단계: 시크릿 모드에서 테스트

새 시크릿 창에서 admin-dashboard.html 접속

5단계: 예상 로그 확인
