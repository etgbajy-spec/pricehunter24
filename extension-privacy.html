Chrome 확장 프로그램 개인정보처리방침

시행일: 2026년 7월 7일 · PriceHunter — 구매판단 리포트 확장

1. 개요

본 방침은 PriceHunter(프라이스헌터) Chrome 확장 프로그램(이하 「확장」)이 처리하는 정보에 대해 설명합니다. 웹사이트 전체 방침은 개인정보처리방침을 참고하세요.

2. 수집하는 정보

확장은 사용자가 「구매판단 의뢰」 버튼 또는 팝업에서 의뢰를 시작할 때만 아래 정보를 읽습니다.

  • 상품 페이지 URL
  • 상품명
  • 표시 가격
  • 사용자가 선택한 옵션

쇼핑몰 페이지를 연 것만으로 위 정보를 서버에 자동 전송하지 않습니다.

3. 브라우저 내 저장

연결 사이트 설정(운영 사이트 / 로컬 개발)만 Chrome storage API로 저장합니다. 개인 식별 정보는 저장하지 않습니다.

4. 정보 이용 목적

  • PriceHunter 의뢰 페이지(request-lite 등) 자동 입력
  • 사용자가 의뢰를 제출할 때 pricehunt24.com으로 전달

광고, 신용평가, 제3자 판매 목적으로 사용하지 않습니다.

5. 권한

  • activeTab: 사용자가 클릭한 현재 탭에서만 상품 정보 읽기
  • storage: 연결 사이트 설정 저장
  • 호스트 권한: 지원 쇼핑몰(쿠팡·네이버·11번가·G마켓·옥션 등) 및 pricehunt24.com
